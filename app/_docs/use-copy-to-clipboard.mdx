## About

The **useCopyToClipboard** hook provides a simple way to copy text to the clipboard with visual feedback. It includes fallback support for older browsers that don't support the modern Clipboard API, ensuring compatibility across different environments.

### Features

- ✅ **Modern Clipboard API**: Uses the native `navigator.clipboard.writeText()` when available
- ✅ **Fallback Support**: Automatically falls back to `document.execCommand('copy')` for older browsers
- ✅ **Visual Feedback**: Built-in `isCopied` state for user feedback
- ✅ **Customizable Delay**: Configure how long the "copied" state persists
- ✅ **Type Safe**: Full TypeScript support
- ✅ **Zero Dependencies**: No external dependencies required

## Usage

### Basic copy button

<CodeBlock contentClassName="h-fit">

```tsx
import { useCopyToClipboard } from "@/hooks/use-copy-to-clipboard";
import { Button } from "@/components/ui/button";

export function CopyButton() {
  const { isCopied, copyToClipboard } = useCopyToClipboard();

  const handleCopy = () => {
    copyToClipboard("https://example.com");
  };

  return (
    <Button onClick={handleCopy} variant="outline">

    </Button>
  );
```

### Copy with toast notification

<CodeBlock contentClassName="h-fit">

```tsx
import { useCopyToClipboard } from "@/hooks/use-copy-to-clipboard";
import { Button } from "@/components/ui/button";
import { toast } from "sonner";
import { useEffect } from "react";

export function CopyWithToast() {
  const { isCopied, copyToClipboard } = useCopyToClipboard();

  useEffect(() => {
    if (isCopied) {
      toast.success("Copied to clipboard!");
    }
  }, [isCopied]);

  const handleCopy = () => {
    copyToClipboard("Text copied with toast notification!");
  };

  return (
    <Button onClick={handleCopy}>
      Copy with Toast
    </Button>
  );
}
```

## API

### Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `delay` | `number` | `5000` | Duration in milliseconds for how long `isCopied` remains `true` |

### Returns

| Property | Type | Description |
|----------|------|-------------|
| `isCopied` | `boolean` | Whether text was recently copied (true for `delay` milliseconds after copying) |
| `copyToClipboard` | `(text: string) => void` | Function to copy text to clipboard |

### Browser Compatibility

The hook automatically detects browser capabilities and uses the appropriate method:

- **Modern browsers**: Uses `navigator.clipboard.writeText()` (preferred method)
- **Older browsers**: Falls back to `document.execCommand('copy')` with textarea approach
- **Error handling**: Catches and logs errors when copy operations fail
